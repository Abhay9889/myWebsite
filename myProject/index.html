HTML Tutorials

Master HTML with comprehensive tutorials covering all aspects of web structure and interactivity.

Introduction to HTML

HTML (HyperText Markup Language) is the backbone of web development, defining the structure of webpages using tags and attributes.

<!DOCTYPE html>
<html lang="en">
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <p>Welcome to HTML!</p>
  </body>
</html>

HTML Structure

Every HTML document follows a standard structure with DOCTYPE, html, head, and body elements.

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Page</title>
  </head>
  <body>
    <header><h1>Hello Krishna</h1></header>
    <main><p>Main content</p></main>
    <footer><p>Footer</p></footer>
  </body>
</html>

HTML Forms

Forms collect user input with various elements like text fields, buttons, and dropdowns.

<form action="/submit" method="POST">

HTML Canvas

The canvas element allows dynamic, scriptable rendering of 2D shapes and bitmap images.

<canvas id="myCanvas" width="200" height="100"></canvas>
<script>
  let canvas = document.getElementById("myCanvas");
  let ctx = canvas.getContext("2d");
  ctx.fillStyle = "blue";
  ctx.fillRect(20, 20, 50, 50);
  ctx.strokeStyle = "black";
  ctx.strokeRect(80, 20, 50, 50);
</script>

HTML SVG

Scalable Vector Graphics (SVG) create vector-based graphics that scale without losing quality.

<svg width="100" height="100">
  <circle cx="50" cy="50" r="40" stroke="black" stroke-width="2" fill="red" />
  <rect x="10" y="10" width="30" height="30" fill="blue" />
</svg>

HTML Accessibility

Accessibility ensures webpages are usable by everyone, including those with disabilities.

HTML5 APIs

HTML5 introduces APIs like Geolocation, Drag and Drop, and Web Storage.

<button onclick="getLocation()">Get Location</button>
<div id="locationOutput"></div>
<script>
function getLocation() {
  if (navigator.geolocation) {
    navigator.geolocation.getCurrentPosition(showPosition);
  } else {
    document.getElementById("locationOutput").innerHTML = "Geolocation not supported.";
  }
}
function showPosition(position) {
  document.getElementById("locationOutput").innerHTML = 
    "Latitude: " + position.coords.latitude + 
    "<br>Longitude: " + position.coords.longitude;
}
</script>

Data Attributes

Custom data attributes store extra information on HTML elements.

<div id="product" data-id="123" data-category="electronics">Product Name</div>
<script>
  let product = document.getElementById("product");
  console.log(product.dataset.id); // 123
  console.log(product.dataset.category); // electronics
</script>
